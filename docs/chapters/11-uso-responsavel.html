ImportanteReflexão final

Dados são instrumentos de poder. Quem os produz, quem os acessa e como são interpretados definem quais problemas entram na agenda pública de saúde e quais populações se tornam invisíveis.

O profissional de saúde crítico é um agente fundamental na produção e no uso ético dos dados em saúde.

## As cinco práticas essenciais

```{r}

praticas <- tibble::tribble(
  ~`#`, ~Prática, ~`Por que é importante`,

  "5", "Considere a série histórica", "Um ponto isolado é pouco informativo. Tendências ao longo de anos revelam padrões mais confiáveis."

tabela_indicador(praticas)
```

## Checklist para análise de indicadores

Antes de interpretar qualquer indicador, pergunte-se:

### Sobre os dados

- [ ] Qual é a **fonte** dos dados? É confiável e auditável?
- [ ] Qual é o **período** de referência? Os dados são atuais?
- [ ] Qual é a **cobertura** do sistema de informação? Há subregistro?
- [ ] Como os dados foram **coletados**? Há viés de seleção?

### Sobre o indicador

- [ ] O indicador é uma **proporção, taxa, razão ou índice**?
- [ ] O **denominador** é adequado? Quem está incluído e quem está excluído?
- [ ] A taxa é **bruta ou padronizada**? Faz sentido comparar diretamente?
- [ ] Os dados estão **desagregados** por sexo, idade, raça/cor, região?

### Sobre a interpretação

- [ ] Estou atribuindo **causalidade** a uma simples correlação?
- [ ] Estou cometendo **falácia ecológica** (inferindo indivíduos a partir de grupos)?
- [ ] A tendência é real ou resultado de **mudança metodológica**?
- [ ] Quem **se beneficia** desta interpretação? Quem é **invisibilizado**?

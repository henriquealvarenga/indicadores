1 Como Usar Indicadores com Responsabilidade

Diante das limitações apresentadas, o uso responsável de indicadores de saúde requer uma postura crítica, metodologicamente informada e eticamente comprometida.

1.1 As cinco práticas essenciais

Mostrar código R 
praticas <- tibble::tribble(
  ~`#`, ~Prática, ~`Por que é importante`,
  "1", "Triangule fontes",        "Nunca baseie decisões em um único indicador. Use múltiplas fontes e indicadores complementares.",
  "2", "Desagregue os dados",     "Questione o que está escondido na média. Analise por raça/cor, sexo, região, renda e faixa etária.",
  "3", "Avalie a qualidade",      "Verifique: cobertura do sistema, taxa de incompletude, comparabilidade metodológica e período.",
  "4", "Correlação ≠ Causalidade", "Indicadores descrevem padrões. Causalidade exige desenhos de estudo adequados e análise de confundidores.",
  "5", "Considere a série histórica", "Um ponto isolado é pouco informativo. Tendências ao longo de anos revelam padrões mais confiáveis."
)

tabela_indicador(praticas)

1.3 Indicadores como instrumentos de poder

Mostrar código R 
poder <- tibble::tribble(
  ~Dimensão, ~Pergunta, ~Implicação,
  "Produção",      "Quem coleta os dados?",        "Comunidades sem UBS não geram dados — são invisíveis",
  "Acesso",        "Quem pode usar os dados?",      "Dados abertos são condição para controle social",
  "Interpretação", "Quem analisa e comunica?",       "A narrativa sobre os dados molda políticas públicas",
  "Ação",          "Quem decide com base nos dados?", "Indicadores sem ação política não salvam vidas"
)

tabela_indicador(poder, caption = "Dados são instrumentos de poder")
ImportanteReflexão final

Dados são instrumentos de poder. Quem os produz, quem os acessa e como são interpretados definem quais problemas entram na agenda pública de saúde e quais populações se tornam invisíveis.

O profissional de saúde crítico é um agente fundamental na produção e no uso ético dos dados em saúde.
